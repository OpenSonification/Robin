How to use it

Arrow keys
Move around the grid from anywhere on the page.
S C T D
Select square, circle, triangle, or diamond.
Shift
Add the selected shape to the current cell.
Shift + arrows
Add the selected shape while moving.
Backspace
Remove every shape from the current cell.
1
Play the current row from left to right.
2
Play the current column from bottom to top.
3
Play only plotted shapes, one column at a time.
4
Play only plotted shapes, one row at a time.
Tab
Move between the controls on the left.
Enter / Space
Use the button that currently has focus.

Headphones make the left-to-right movement easiest to hear.
